📋 Overview<br>
This repository provides supplementary materials to accompany our manuscript. The hard-carbon dataset includes microstructural parameters and corresponding sodium storage performance metrics, enabling the reproduction and validation of our machine learning models.
<br>
📁 Repository Structure<br>
├── data/<br>
│   └── datasetclean.csv    # Complete dataset<br>
├── configs/<br>
│   ├── TabPFN.py      # TabPFN optimal hyperparameters<br>
│   ├── RF.json        # Random Forest optimal hyperparameters<br>
│   ├── XGB.py         # XGBoost optimal hyperparameters<br>
│   └── LGBM.py        # LightGBM optimal hyperparameters<br>
└── README.md                      # This file<br>

📊 Dataset Description<br>
The dataset contains experimental measurements of biomass-derived hard carbon materials.

⚙️ Hyperparameter Configurations<br>
The optimal hyperparameters for each selected model are provided in the configs/ directory.
